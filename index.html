00 Sambungkan Spreadsheet

Data upload akan tersimpan permanen di Google Sheet kamu sendiri — bukan cuma di browser ini.

Belum tersambung
Belum punya spreadsheet-nya? Klik di sini untuk cara setup (sekali saja)
  1. Buka sheets.google.com → buat spreadsheet baru (boleh kosong, boleh dikasih nama "Database Dashboard Hypospot").
  2. Di menu atas klik Extensions → Apps Script.
  3. Hapus semua kode default di editor yang terbuka, lalu paste kode di bawah ini (klik tombol "Salin Kode").
  4. Klik ikon 💾 Save, lalu klik Deploy → New deployment.
  5. Pilih tipe Web app. Isi "Execute as" = Me, "Who has access" = Anyone. Klik Deploy.
  6. Google akan minta izin akses — pilih akun kamu, klik Advanced → Go to (nama project) → Allow.
  7. Salin URL yang muncul (diakhiri /exec), paste ke kolom di atas, lalu klik Sambungkan.
  8. Kalau nanti kode Apps Script diubah lagi, ingat klik Deploy → Manage deployments → Edit (pensil) → New version → Deploy supaya perubahan aktif.
