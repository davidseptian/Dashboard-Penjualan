Dashboard Penjualan per Produk

Kelola data Pesanan, Income, dan Pemetaan Produk, lalu pantau performa penjualan — semua dalam satu tempat.

00 Sambungkan Spreadsheet

Data upload akan tersimpan permanen di Google Sheet kamu sendiri — bukan cuma di browser ini.

Belum tersambung
Belum punya spreadsheet-nya? Klik di sini untuk cara setup (sekali saja)

⚠️ Kalau kamu sudah pernah setup versi sebelumnya: cukup ganti kode Apps Script-nya dengan yang baru di bawah (Deploy → Manage deployments → Edit → New version), spreadsheet & data lama kamu tidak akan hilang — cuma nambah 3 sheet baru (Orders, Income, Mapping).

  1. Buka sheets.google.com → buat spreadsheet baru (boleh kosong, boleh dikasih nama "Database Dashboard Hypospot").
  2. Di menu atas klik Extensions → Apps Script.
  3. Hapus semua kode default di editor yang terbuka, lalu paste kode di bawah ini (klik tombol "Salin Kode").
  4. Klik ikon 💾 Save, lalu klik Deploy → New deployment.
  5. Pilih tipe Web app. Isi "Execute as" = Me, "Who has access" = Anyone. Klik Deploy.
  6. Google akan minta izin akses — pilih akun kamu, klik Advanced → Go to (nama project) → Allow.
  7. Salin URL yang muncul (diakhiri /exec), paste ke kolom di atas, lalu klik Sambungkan.
  8. Kalau nanti kode Apps Script diubah lagi, ingat klik Deploy → Manage deployments → Edit (pensil) → New version → Deploy supaya perubahan aktif.

00b Sambungkan Sumber HPP (Master Modal)

HPP tiap SKU diambil langsung dan otomatis dari spreadsheet Master Modal kamu — jadi kalau harga modal berubah di situ, Analisis Laba di dashboard ikut update sendiri.

Belum tersambung

Syarat: sheet-nya harus di-share minimal "Anyone with the link can view" (di tombol Share, kanan atas Google Sheets). Sistem cari kolom SKU dan MODAL / HPP otomatis dari sheet tab yang kamu buka terakhir (ditandai lewat gid di URL).

01 Upload Data Bulanan

Format CSV: kolom Nama Produk, Qty, Sub Total. Nama file idealnya memuat nama bulan (mis. "...FEB.csv" atau "...JANUARI.csv"). Data akan tampil dulu di dashboard — klik "Simpan ke Spreadsheet" kalau sudah yakin mau ditulis permanen.

⬆
Klik atau seret file CSV ke sini
Bisa upload beberapa bulan sekaligus

02 Upload Data Pesanan (per Platform)

Bisa upload file mentah asli (Order_all Shopee / Semua_pesanan TikTok, .xlsx) — otomatis dideteksi platform & bulannya. Atau CSV bersih: No. Pesanan, Nama Produk, Qty, Subtotal Kotor (+ kolom SKU opsional).

⬆
Klik atau seret file Pesanan ke sini (.xlsx mentah atau .csv bersih)
File mentah: 1 file bisa berisi banyak bulan sekaligus, auto-split

03 Upload Data Income (per Platform)

Bisa upload file mentah asli (Income Shopee sheet "Income" / income TikTok sheet "Detail pesanan", .xlsx) — auto-deteksi platform. Atau CSV bersih: No. Pesanan, Total Penghasilan.

⬆
Klik atau seret file Income ke sini (.xlsx mentah atau .csv bersih)
1 file = 1 platform (boleh gabung banyak bulan)
